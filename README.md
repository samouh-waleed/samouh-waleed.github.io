# Personal Portfolio

## Overview
This is a personal portfolio website built using React.js. It showcases skills, projects, and experiences in a visually appealing format with a modern design. The website features a single-page application (SPA) layout with smooth scrolling navigation and an integrated PDF viewer for the resume.

## Features

- **Single-Page Application**: All sections are displayed on one page with smooth scrolling.
- **Modern Design**: A clean and responsive design using CSS and Bootstrap.
- **React-Icons**: Icons from the react-icons library for a visually appealing design.
- **Carousel for Projects**: Custom carousel for project displays using Bootstrap and react-icons.
- **PDF Viewer**: Integrated PDF viewer for displaying the resume.
- **Responsive Layout**: Optimized for viewing on various devices.

## How to Use

1. **Install dependencies**
    ```sh
    npm install
    ```

2. **Run the application**
    ```sh
    npm start
    ```
